Introduction

Candel CSS is a free, open-source CSS framework that provides ready-to-use frontend components to build responsive and modern websites.

Usage

To use Candel CSS in your project:

Option 1: Copy-paste the stylesheet <link> into your <head> before all other stylesheets.

Option 2: Copy-paste the @import into your stylesheets.

Download

You can download the ready-to-use compiled code for Candel CSS and easily it use it in your project.

Typography

Candel CSS uses Nunito Sans as its default font, which is a free font available on Google Fonts.

Contribute

To keep up with today's design requirements, we require your help too.
Any contribution / review / suggestion is highly appreciated.
